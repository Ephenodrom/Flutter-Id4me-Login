## Install

### pubspec.yaml

Update pubspec.yaml and add the following line to your dependencies.

```yaml
dependencies:
  flutter_id4me_login: ^1.0.0
```

## Import

Import the package with :

```dart
import 'package:flutter_id4me_login/flutter_id4me_login.dart';
```
